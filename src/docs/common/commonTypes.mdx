# Common Types and Props
We have a common set of props that are supported by a number of components. These will be mentioned in the docs for the individual components as well. We present them below with a reference component.

Before we do that, we present some common types referenced by the props.

## CSSLength
`type CSSLength = number | string`. 

Examples: 
* 5 (implies 5px)
* '5rem'

## CSSBox

Various versions of providing common shorthand properties

```ts
export type CSSBox =
  /** top,right,left,bottom */
  | CSSLength
  /** [top & bottom, left & right] */
  | [CSSLength, CSSLength]
  /** [top,right,bottom,left] */
  | [CSSLength, CSSLength, CSSLength, CSSLength]
```

e.g. `5` (means `5px` top,right,left,bottom), `['5rem','10px']` (means `5rem` top & bottom, `10px` left & right), `[10,5,'10rem', 15]` (means `10px` top, `5px` right, `10rem` bottom, `15px` left).

## SpacingProp

```ts
export interface SpacingProp {
  /** Spacing between each child */
  spacing?: CSSLength,
}
```

* `spacing?: CSSLength` Controls spacing between children ([concepts covered in spacing principle][principle-spacing]). e.g. `spacing={10}` gives us:

## SizeProps

The size props (`height,minHeight,maxHeight,width,minWidth,maxWidth`) allow for explicit size control as covered in the [sizing principle][principle-sizing] section.

## SizingProp

The `sizing` prop is covered in the [sizing principle][principle-sizing] section.

## PaddingProp

```ts
/** 
 * Add support for padding
 */
export interface PaddingProp {
  padding?: CSSBox,
}
```

Allows you to specifiy padding as a [CSSBox](#cssbox). Using padding effectively is covered in the [spacing principle][principle-spacing] section.

## ScrollProp 

Contains `scroll?` member. Type definition and effective use is coverd in the [scrolling principle][principle-scrolling] section.

## BaseProps
Objective: A consistent set of customization options. Collects the following props into a common type:

Members:

* `PaddingProp`: [Specifies padding](#paddingprop).
* `ScrollProp`: [Specify scrolling behaviours](scrollprop).
* `styles?: (typestyle.types.NestedCSSProperties | falsy)[]`: an array of custom typestyle `style` objects. Any falsy array members will be safely ignored.

From react: 
* `className?: string`: css class name.
* `style?: React.CSSProperties`: any custom react `style` you want to pass in.

For the tag control: 
```ts
tag?: {
  name?: 'div' /** default */ | 'a' | 'button' // etc.
  props?: React.HTMLProps // props passed to the tag
}
```
