## Spacing

You can control the spacing between children with the spacing prop:
```ts
  spacing?:
    | CSSLength
    | [/** Vertical */ CSSLength, /** Horizontal */ CSSLength]
```
Example (vertical:10, horizontal:20): 

<Playground>
