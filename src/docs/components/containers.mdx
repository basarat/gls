# Containers

We have two general purpose containers, `Flex` and `Content`. The ideas of *flex* and *content* are explained in the [sizing principle][principle-sizing]. If you have more than one child item, you probably want to use something like a [Vertical][vertical] or [Horizontal][horizontal].
