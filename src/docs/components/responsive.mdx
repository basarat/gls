# Responsive 

Lays out children in a vertically or horizontally spaced layout based on given breakpoint.

<Playground>
  <Responsive>
    <div style={{backgroundColor: "lightskyblue"}}>Hello</div>
    <div style={{backgroundColor: "lightpink"}}>World</div>
  </Responsive>
</Playground>

## GLSProps

`Responsive` [supports all `GLSProps`][common-types].

## ChildPlacementProps

`Responsive` [supports all `ChildPlacementProps`][common-types].

## SizingProp

`Responsive` supports the `sizing` prop as explained in the [sizing principle][principle-sizing].
