# Vertical 

Lays out children in a vertically spaced layout.

<Playground>
  <Vertical>
    <div style={{backgroundColor: "lightskyblue"}}>Hello</div>
    <div style={{backgroundColor: "lightpink"}}>World</div>
  </Vertical>
</Playground>

Since `Vertical` follows the [spacing principal][principle-spacing] you can compose / nest `Vertical`s easily. The following example has three `Verticals` in a `Vertical(Vertical(blue,pink),Vertical(blue,pink))` heirarchy but to the user it looks seemless. This ability to nest allows you to easily create and reuse components without having to worry about how they will get used.

<Playground>
  <Vertical>
    <Vertical>
      <div style={{backgroundColor: "lightskyblue"}}>Hello</div>
      <div style={{backgroundColor: "lightpink"}}>World</div>
    </Vertical>
    <Vertical>
      <div style={{backgroundColor: "lightskyblue"}}>Second Hello</div>
      <div style={{backgroundColor: "lightpink"}}>Second World</div>
    </Vertical>
  </Vertical>
</Playground>

## Props
* `spacing?: BoxUnit` Controls spacing between children ([concepts covered in spacing principle][principle-spacing]).

<Playground>
  <Vertical spacing={10}>
    <div style={{backgroundColor: "lightskyblue"}}>Hello</div>
    <div style={{backgroundColor: "lightpink"}}>World</div>
    <div style={{backgroundColor: "darkorange"}}>Again</div>
  </Vertical>
</Playground>

* `verticalAlign?: 'top' /** default */ | 'center' | 'bottom'` Controls vertical alignment of children.

<Playground>
  <Vertical verticalAlign='bottom' style={{height: '100px'}} spacing={10}>
    <div style={{backgroundColor: "lightskyblue"}}>Hello</div>
    <div style={{backgroundColor: "lightpink"}}>World</div>
  </Vertical>
</Playground>

* `horizontalAlign?: 'left' /** default */ | 'center' | 'right'` Controls horizontal alignment of children.

<Playground>
  <Vertical horizontalAlign='right'>
    <button>Hello</button>
    <button>World</button>
  </Vertical>
</Playground>

## GLSProps

`Vertical` [supports all `GLSProps` e.g. `padding`][common-types]: 

<Playground>
  <Vertical padding={20}>
    <div style={{backgroundColor: "lightskyblue"}}>Hello</div>
    <div style={{backgroundColor: "lightpink"}}>World</div>
  </Vertical>
</Playground>

## SizingProp

`Vertical` supports the `sizing` prop as explained in the [sizing principal][principle-sizing].
